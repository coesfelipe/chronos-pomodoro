# Chronos Pomodoro

Chronos Pomodoro é uma aplicação web desenvolvida com React e Vite que implementa a técnica Pomodoro. Ela permite gerenciar ciclos de estudo com temporizadores para foco, descanso curto e descanso longo, proporcionando uma interface simples e intuitiva para aumentar a produtividade e a organização durante os estudos.

# Tecnologias: 
React, Vite, JavaScript/TypeScript, HTML5 e CSS3.

# Funcionalidades:
⏱️ Cronômetro para sessões de foco.
☕ Descanso curto.
🌙 Descanso longo.
🔄 Alternância automática entre os ciclos.
🎨 Interface moderna e responsiva.
